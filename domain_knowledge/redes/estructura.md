# Estructura de una Red

## Descripcion

Cada red posee una estructura interna que replica a nivel local las funciones de la iglesia.

## Componentes

### 1. Lider de red

- Designado por el cuerpo pastoral (pastor y esposa).
- Supervisa todas las casas de paz.
- Acompana a los lideres de cdP.
- Autonomo, no se relaciona con otros lideres de red en trabajo.
- Periodo indefinido.

### 2. Representantes departamentales

Cada red cuenta con 4 representantes:

| Departamento | Funcion |
|-------------|---------|
| **Evangelismo** | Ejecuta estrategias evangelisticas en la red. |
| **Afirmacion** | Gestiona integracion de nuevos asistentes. |
| **Discipulado** | Coordina la formacion espiritual. |
| **Envio** | Control de listas y preparacion para envio. |

### 3. Casas de paz

- Unidades basicas que conforman la red.
- No hay limite de casas de paz por red.

### 4. Sublideres (no oficiales)

Algunas redes tienen sublideres de red como colaboradores cercanos, pero de manera no oficial.

## Organigrama conceptual

```
Cuerpo Pastoral (Pastor y esposa)
         |
    Lider de Red  (periodo indefinido)
         |
    +----+----+
    |    |    |
    Ev  Af  Di  En  (4 representantes)
    |
    CDP 1   CDP 2   CDP N
    Lider   Lider   Lider
```

## Casos especiales

- Una casa de paz puede cambiar de red. Se documenta el cambio, no los motivos.
- Puede haber lider de red interino (tecnico, se documenta).
