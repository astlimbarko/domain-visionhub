# Software Spec

Documentacion de referencia para el desarrollo del software VisionHub.

Este folder contiene la descripcion de los modulos, funcionalidades yAreas pendientes de revision. Esta pensado para ser llevado a Claude Code y generar los archivos spec tecnicos necesarios.

## Contenido

- [modulos.md](modulos.md) — Descripcion de los modulos del software, foco en Modulo 1.
- [bd-modelo.md](bd-modelo.md) — Recomendaciones de base de datos: modelo de personas, nombres, apellido casada, mapeo de formularios, enums, reglas de moneda, configuracion de iglesias (tenancy).
- [iu-front.md](iu-front.md) — Interfaz de usuario (frontend). Contenedor para documentar la IU cuando este disponible.
- [dashboards/](dashboards/) — Descripcion de los dashboards por rol: que datos ven y que graficos usar.
- [pendientes.md](pendientes.md) — Areas con informacion incompleta o que requieren revision adicional.

## Fuente

La informacion se infirio del archivo `uptdate 13-7-26-a.txt` (modelado CHMR) y de la documentacion existente en `domain_knowledge/`.
