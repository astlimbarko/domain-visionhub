# Interfaz de Usuario — VisionHub (Frontend)

## Proposito

Este archivo es el contenedor para la documentacion de la interfaz de usuario (frontend) de VisionHub. Cuando el archivo de la IU este disponible, se insertara aqui o se linkara desde este documento.

## Estructura esperada

Una vez que llegue el archivo de la IU, se documentara:

- Layouts y pantallas por plataforma (web, movil)
- Componentes UI reutilizables
- Flujo de navegacion por rol
- Formularios y validaciones visuales
- Responsive design / adaptacion movil
