# Dashboard — Sublider de Casa de Paz

El sublider puede ver un dashboard SIMILAR al del lider, pero con alcance limitado por configuracion del Supervisor.

---

## Que puede ver (configurable desde Supervision)

El Supervisor decide que secciones ve el sublider. Por defecto:

| Seccion | Visible por defecto | Que ve |
|---------|---------------------|--------|
| **Formulario de reporte** | Si | Puede llenar el reporte semanal |
| **Asistencia** | Si | Lista de asistentes (solo ver, no editar) |
| **Miembros** | Si | Lista basica de miembros (nombre, estado) |
| **Ofrendas** | No | El Supervisor decide si ve montos |
| **Graficos** | No | El Supervisor decide si ve graficos |
| **Historial** | No | El Supervisor decide si ve semanas pasadas |

### El Supervisor puede configurar:

- Si el sublider ve la seccion de ofrendas (montos).
- Si el sublider ve graficos de asistencia.
- Si el sublider ve el historial de reportes anteriores.
- Si el sublider puede editar el reporte despues de enviarlo.

---

## Formulario de reporte

El sublider tiene acceso al **mismo formulario** que el lider:
- Puede llenar y enviar el reporte semanal.
- Despues de enviar, no puede editarlo (a menos que el Supervisor lo active).

---

## Diferencia con el Lider

| Funcion | Lider | Sublider |
|---------|-------|----------|
| Llenar reporte | Si | Si (configurable) |
| Ver asistencia | Si | Si |
| Ver ofrendas | Si | Configurable |
| Ver graficos | Si | Configurable |
| Editar miembros | Si | No |
| Ver historial completo | Si | Configurable |
| Recibir notificaciones | Si | Configurable |
